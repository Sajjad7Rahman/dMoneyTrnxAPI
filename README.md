# dMoneyTrnxAPI

## Project Summary:
* This Project has been automated Transaction API from POSTMAN using Newman

## Technology used:
* Postman
* Newman
* VS code

## Pre-requisite/s:
* NodeJs

## How to run:
``` git clone ```
``` npm i ```
``` npm test ```
A Report will be generated in you project source root
